## ⚙️ Example Profile Configuration

Set up your personal staking and bridging settings easily in `bridge-profile.yaml`:

    username: SatoshiDreamer
    preferred_language: en
    networks:
      - bitcoin
      - starknet
      - ethereum
    staking:
      minimum_hold_days: 7
      auto_compound: true
      preferred_yield_token: sBTC
    api_integrations:
      openai_enabled: true
      claude_enabled: true
    notification:
      method: email
      address: you@example.com
    security:
      2fa: true
      backup_email: recovery@protonmail.com

## 💻 Console Usage Examples

**Stake Bitcoin and Bridge to Starknet**

    btc-bridge stake --amount 0.5 --from bitcoin --to starknet

**Get AI Portfolio Insights**

    btc-bridge insights --address 1YourBTCAddrHere --provider openai

**Bridge Assets Cross-Chain**

    btc-bridge bridge --amount 1.2 --from starknet --to ethereum --token sBTC

**Switch Interface Language to Japanese**

    btc-bridge config --set preferred_language=ja
